# DocAnalyzer

DocAnalyzer is a Python application for analyzing text documents and books across multiple file formats. The project began as the BookBot project from Boot.dev and has since expanded with additional features and support for multiple document formats.

## Features

- Analyze TXT files
- Analyze PDF documents
- Analyze Microsoft Word files
- Analyze EPUB eBooks
- Analyze Markdown (.md) files
- Analyze HTML files

For each document, DocAnalyzer provides:
- Total word count
- Character frequency analysis
- Sorted character statistics
- Reading report in the terminal

## Supported Formats

- TXT
- PDF
- DOCX
- EPUB
- Markdown
- HTML
